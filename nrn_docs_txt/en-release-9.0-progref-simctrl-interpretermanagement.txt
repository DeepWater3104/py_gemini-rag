quit
(
)

Python
Exits the program. Can be used as the action of a button. If edit buffers
are open you will be asked if you wish to save them before the final exit.
HOC
Exits the program. Can be used as the action of a button. If edit buffers
are open you will be asked if you wish to save them before the final exit.
MATLAB
Syntax:
n.quit()
Description:
Exits the program and MATLAB.
